# 크론잡 메시지 포맷 가이드

> 원본: openclaw-config/cron/FORMAT_GUIDE.md 기반. 변경 시 원본과 동기화 필요.

## 기본 규칙

모든 scheduled task 출력물은 아래 포맷을 준수한다.

### 1. 상단 배너 (필수)

```
━━━━━━━━━━━━━━━━━━━━━━━━
[emoji] [YYYY-MM-DD(요일) HH:MM] [태스크명]
━━━━━━━━━━━━━━━━━━━━━━━━
```

### 2. 섹션 구분선

- `---` 사용 금지 (Discord/Slack에서 수평선 미렌더링)
- 대신 유니코드 구분선 사용: `━━━━━━━━━━━━━━━━━━━━━━━━` (24자)
- 모바일 화면 줄바꿈 방지를 위해 24자 이하로 제한

### 3. 태스크별 배너 헤더

| 태스크 | 배너 |
|--------|------|
| 아침 브리핑 | `☀️ [날짜 05:30] 아침 브리핑` |
| 테크 뉴스 | `📰 [날짜 07:00] 테크 뉴스 브리핑` |
| 바이브 코딩 | `🤖 [날짜 08:00] 바이브 코딩 위클리` |
| 하프타임 (오전) | `⏸️ [날짜 11:00] 하프타임 점검` |
| 하프타임 (오후) | `⏸️ [날짜 15:00] 하프타임 점검` |
| 경제/부동산 뉴스 | `🏠 [날짜 18:00] 경제/부동산 뉴스 브리핑` |
| 저녁 브리핑 | `🌙 [날짜 22:00] 저녁 브리핑` |
| 해외투자 뉴스 | `📈 [날짜 07:30] 해외투자 뉴스 브리핑` |

## 적용 예시

```
━━━━━━━━━━━━━━━━━━━━━━━━
📰 [2026-04-05(토) 07:00] 테크 뉴스 브리핑
━━━━━━━━━━━━━━━━━━━━━━━━

**오늘의 핫토픽**
> Claude Code 2.2 릴리즈 ...

━━━━━━━━━━━━━━━━━━━━━━━━

**주요 뉴스**
- **[제목]** - 한 줄 요약 (출처, 링크)
```
