# mc-cc

Claude Code development workflow skills - commit, PR, review, DB, deployment, plugin creation, and more.

## Skills

| 스킬 | 설명 |
|------|------|
| `/commit` | 변경사항을 분석하고 컨벤션에 맞는 커밋을 생성한다 |
| `/db-run` | 개발 DB에 SQL 쿼리를 실행한다 |
| `/feedback-pr` | PR을 리뷰하고 파일별 인라인 코멘트로 피드백을 작성한다 |
| `/find-aws-logs` | ECS 서버의 CloudWatch 로그를 검색한다 |
| `/hook-creator` | Claude Code hook을 생성하고 설정한다 |
| `/pluginize` | 프로젝트의 skills, agents, commands를 Claude Code Plugin으로 변환한다 |
| `/pr` | PR을 생성한다 |
| `/push` | 스모크 테스트 실행 후 코드를 푸시한다 |
| `/push-n-pr` | 푸시 + PR 생성을 일괄 수행한다 |
| `/retro` | 기간별 작업 회고를 자동 생성한다 |
| `/review-plan` | 구현 계획을 staff engineer 관점에서 리뷰한다 |
| `/review-pr` | PR의 변경사항을 리뷰한다 |
| `/skill-creator` | 새로운 skill을 생성한다 |
| `/slash-command-creator` | 슬래시 커맨드를 생성한다 |
| `/subagent-creator` | 서브에이전트를 생성한다 |
| `/ticket-destroyer` | 이슈 분석, 계획 수립 또는 코드 수정 및 PR 생성 |
| `/youtube-collector` | 유튜브 채널 영상 수집 및 자막 기반 요약 |

## Agents

| 에이전트 | 설명 |
|----------|------|
| `brand-logo-finder` | Brandfetch를 활용한 브랜드 로고 검색 |
| `dev-responder` | 개발 조치 사항 분석 및 대응 방안 제시 |

## Commands

| 커맨드 | 설명 |
|--------|------|
| `/db-query` | SQL 쿼리 파일을 생성한다 |
| `/figma-spec` | Figma 디자인을 분석하여 백엔드 개발 명세서를 생성한다 |
